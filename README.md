# Amol Raut — Developer Portfolio

A modern, responsive developer portfolio built with **React + Vite + Tailwind CSS**, featuring smooth animations, project showcases, and a fully functional contact form.

🔗 **Live Demo:** [your-portfolio-link.vercel.app](https://your-portfolio-link.vercel.app)

| Layer | Technology |

## Getting Started

### 1. Clone & Install
```bash
git clone https://github.com/your-username/my-portfolio.git
cd my-portfolio
npm install
```

### 2. Set Up Environment Variables

Create a `.env` file in the root directory:
```env
VITE_SERVICE_ID=your_service_id
VITE_TEMPLATE_ID=your_template_id
VITE_PUBLIC_KEY=your_public_key
```

> ⚠️ Restart the dev server after adding environment variables.

### 3. Run Locally
```bash
npm run dev
```

### 4. Build for Production
```bash
npm run build
npm run preview
```

## Contact Form Setup (EmailJS)

This project uses [EmailJS](https://www.emailjs.com/) to send messages directly from the frontend — no backend required.

1. Create a free account at [emailjs.com](https://www.emailjs.com/)
2. Set up an **Email Service**, a **Contact Template**, and an **Auto-Reply Template**
3. Copy your credentials into the `.env` file as shown above

---

## Deployment (Vercel)

1. Push the project to GitHub
2. Import the repo at [vercel.com](https://vercel.com)
3. Add your environment variables in the Vercel dashboard
4. Hit **Deploy** 🚀
